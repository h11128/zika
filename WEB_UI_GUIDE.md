# 🀄 Chinese Learning Cards Generator - Web UI 使用指南

## 🚀 快速开始

### 启动应用
```bash
python -m streamlit run web_ui.py
```

然后在浏览器中打开 `http://localhost:8501`

## 📱 界面介绍

### 主界面布局
- **左侧面板**: 输入控制和参数设置
- **右侧面板**: 实时预览
- **侧边栏**: 使用统计和快速链接

### 侧边栏功能
- **📊 使用统计**: 显示内置词典大小和累计生成卡片数
- **📥 导出历史**: 显示最近5次导出记录
- **🔗 快速链接**: 项目相关链接

## 📝 输入方式

### 1. 手动输入
- **模板选择**: 提供8种预设模板
  - 基础汉字: 爱 家 朋友 水 火 山 月 日 木
  - 数字: 一 二 三 四 五 六 七 八 九 十
  - 颜色: 红 黄 蓝 绿 黑 白 灰 粉 紫 橙
  - 家庭: 爸爸 妈妈 哥哥 姐姐 弟弟 妹妹 爷爷 奶奶 外公
  - 身体部位: 头 眼 耳 鼻 嘴 手 脚 心 身体
  - 动物: 猫 狗 鸟 鱼 马 牛 羊 猪 鸡
  - 食物: 米 面 肉 鱼 蛋 奶 茶 水 糖
- **文本输入**: 在文本框中输入空格分隔的汉字
- **🔤 自动分隔按钮**: 点击可将文本框中的连续汉字自动分词
- **连续汉字输入**: 专门的输入框用于输入连续汉字，支持实时分词预览

### 2. CSV文件上传
- **支持格式**: CSV文件
- **列名识别**: 
  - 汉字: `hanzi`, `chinese`, `word`
  - 拼音: `pinyin`, `pronunciation`
  - 英文: `english`, `translation`, `meaning`
- **文件预览**: 上传后显示文件内容预览

## ⚙️ 参数设置

### 基础选项
- **自动生成拼音**: 为缺失拼音的汉字自动生成
- **自动生成翻译**: 为缺失翻译的汉字自动查找
- **页面尺寸**: A4 或 Letter
- **卡片大小**: 4.0-8.0cm，默认5.5cm

### 高级选项（折叠面板）
- **卡片间距**: 0.2-1.0cm，默认0.5cm
- **页面边距**: 0.5-2.0cm，默认1.0cm
- **字体大小**:
  - 汉字: 24-72pt，默认48pt
  - 拼音: 12-36pt，默认18pt
  - 英文: 8-24pt，默认14pt

## 👀 预览功能

### 实时预览
- **3×3网格**: 显示9个卡片位置
- **空卡片占位**: 未填充位置显示虚线边框
- **卡片计数**: 显示总卡片数和页数

### 卡片编辑
- **编辑面板**: 可展开的卡片编辑区域
- **分标签编辑**: 每张卡片一个标签页
- **实时更新**: 编辑后立即更新预览

## 📥 导出功能

### 支持格式
- **PPTX**: 可编辑的PowerPoint格式
- **PDF**: 打印就绪的PDF格式

### 导出流程
1. 点击"导出 PPTX"或"导出 PDF"按钮
2. 系统生成文件（显示进度条）
3. 出现"下载"按钮
4. 点击下载按钮保存文件

### 文件命名
- 格式: `chinese_cards_{卡片数量}.{格式}`
- 示例: `chinese_cards_9.pptx`

## 🔤 自动分词功能

### 智能分词算法
- **基于jieba分词**: 使用成熟的中文分词库
- **学习卡片优化**: 针对学习场景优化分词长度
- **去重处理**: 自动去除重复词汇
- **长词拆分**: 超长词汇自动拆分为适合学习的短词

### 分词规则
1. **单字**: 保持不变（如：我、爱、的）
2. **双字词**: 保持完整（如：家人、朋友、学习）
3. **3-4字词**: 保持完整（如：北京大学、人工智能）
4. **长词**: 拆分为2字词组（如：中国人民解放军 → 中国 人民 解放 军）

### 使用方法
1. **文本框分词**: 在主文本框输入连续汉字，点击"🔤 自动分隔"
2. **专用输入框**: 在"连续汉字输入"框输入，查看分词预览，点击"✅ 应用分词"
3. **实时预览**: 输入连续汉字后立即显示分词结果

### 分词示例
- `我爱我的家人朋友们` → `我 爱 的 家人 朋友 们`
- `今天天气很好` → `今天天气 很 好`
- `学习中文很有趣` → `学习 中文 很 有趣`
- `北京大学是一所著名的大学` → `北京大学 是 一所 著名 的 大学`

## ❓ 常见问题

### Q: 为什么有些汉字没有翻译？
A: 内置词典包含常用汉字，不常见的字可能需要手动添加翻译。

### Q: 如何调整卡片在页面上的位置？
A: 通过调整"页面边距"参数来控制卡片在页面上的位置。

### Q: 支持哪些CSV文件格式？
A: 支持标准CSV格式，列名可以是中文或英文，系统会自动识别。

### Q: 生成的文件保存在哪里？
A: 文件会下载到浏览器的默认下载目录。

### Q: 可以同时生成多少张卡片？
A: 理论上没有限制，但建议单次不超过100张以确保性能。
